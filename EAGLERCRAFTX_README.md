# Apollo Client 1.8.8

Apollo Client is a modified Eaglercraft 1.8.8 client. Built for performace, and PvP. 


## Compiling
To compile the client run `./CompileLatestClient.sh` or `./target_teavm_javascript/CompileJS.sh`
If gradle crashes, ususally due to out of memory error, run `./gralde --stop`

## Testing

The client has only been tested using the **JavaScript** version. It has NOT be tested wit **Desktop** nor **WASM-GC**.

## Commiting


As a contributer, you should add your name to `CONTRIBUTORS.md`. You are also **expected** to obey the rules in `COMMIT.md`. It is important to do so to keep the client alive, and clean.
