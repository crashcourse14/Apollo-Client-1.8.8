# <img src="apolloclient/logos/logo.png" width="80" height="80" style="border-radius:20px;">  Apollo Client
Apollo Client is an open-source modified Eaglercraft client that aims to improve performance, gameplay, and an overall more stable way for player to play Eaglercraft.


The project's website is: https://apolloclientmc.vercel.app .

Use `git clone` to check the source locally, or simply download the zip (not recommended).
