# CONTRIBUTING

This goes for all, and for any developer out there who may commit to the client. Whom ever you may be, these rules are expected to be followed when submiting code. Especially if your code/feature could end up as an **official** feature.

## What not to do

- Commiting pladrized code. 
- Not testing your code before commiting.
- Commiting malicious code.
- Not crediting any code not originally made by you.
